# Product Naming: Cosmio

## Naming Strategy (Diamond Framework)

| Item | Details |
|------|---------|
| Goal | Evoke the vastness and mystery of space in a single word; naturally suggest "GitHub contribution visualization tool" |
| Strengths | Artistic space theme, 4 views, animated SVG, open-source |
| Core Message | "Your coding history becomes the cosmos" |
| Brand Personality | Mysterious, awe-inspiring, refined |
| Sound | Soft, deep resonance (L, N, M, S) |

## Final Decision

### Cosmio

- **Etymology**: Cosmos (universe) + -io (modern tech suffix)
- **Pronunciation**: koz-mee-oh (3 syllables, natural flow)
- **Impression**: Soft, warm vastness. Approachable yet modern.

### Rationale

1. **Extensibility** — Contains "Cosmos" itself, making any space theme addition feel natural
2. **Availability** — Available on both npm and GitHub
3. **Memorability** — Intuitive pronunciation, easy to spell after hearing
4. **Brand Fit** — Premium yet accessible tone

## Runner-up Candidates

| Name | Etymology | Impression | Reason Not Selected |
|------|-----------|-----------|-------------------|
| Stellex | Stella (star) + ex (extension) | Cold, sharp mystique | Too technical, lacks artistic sensibility |
| Orbitra | Orbit + ra (resonance) | Dynamic, grand orbit | Good, but more limited theme extensibility than Cosmio |

## Usage

- npm package: `cosmio`
- GitHub repo: `cosmio`
- GitHub Action: `cosmio-action` or `cosmio`
- CLI command: `npx cosmio`
